Опис Проєкту

Проєкт адмін панелі для сайту букінгу котеджів у Альпах під назвою "Vine Hotel" - це система управління бронюваннями та адмініструванням готелю. Вона розроблена з використанням сучасних технологій, таких як бібліотека React для фронтенду та база даних Supabase для зберігання даних.

Основні функціональні можливості цього проекту включають:

  1. Управління котеджами: Адміністратор може додавати, видаляти та редагувати інформацію про котеджі, таку як опис, фотографії, зручності та ціни.
  2. Бронювання: Система дозволяє адміністратору переглядати та керувати всіма бронюваннями котеджів, здійснювати підтвердження та скасування бронювань.
  3. Керування користувачами: Адміністратор може керувати користувачами системи, включаючи додавання нових акаунтів для персоналу готелю та зміну прав доступу.
  4. Статистика та звіти: Система надає зручні засоби для аналізу даних про бронювання, забезпечуючи адміністратора необхідною інформацією для управління готелем та виявлення тенденцій.
  5. Керування цінами та акціями: Адміністратор може встановлювати ціни на різні періоди, включаючи різні сезони та акційні пропозиції, що допомагає оптимізувати прибуток готелю.
